EASYCHARTS

EasyCharts is a set of easy-to-use and flexible java based charting components that can be used to display chart applets in web pages, create charts in java applications, or generate charts as images in application servers using servlets.

EasyCharts supports bar charts, line charts, pie charts, and overlay charts so you can combine any of those chart types in one chart.
